# CoMatch

CoMatch is a web-based co-founder matching platform built for our second-year Team Project.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Java (REST API)
- Database: SQL

## Team Members
- Megan – Frontend & DevOps
- Dhruva – Backend & Project Manager
- Mridhula – Database & Documentation
- Zakaria – UI/UX & Frontend

## Sprint 1 Goal
User can register, login, and create a founder profile.

## Sprint 2 Goal
User can discover profiles, like or pass them, and generate mutual matches.

## Sprint 3 Goal
Let matched founders chat with each other, get reminders when someone messages them, and report or block people if something feels wrong. We also want better match suggestions.
